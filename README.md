### Prerequisites

1. Node
2. Android Studio
3. Text Editor

### Installing

Clone Project and run the following command

```
npm run start
```
The command npm run start will deploy the project in web view

To deploy it in mobile
```
ionic cordova emulate android || ionic cordova emulate ios
```
